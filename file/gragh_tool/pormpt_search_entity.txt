## 輸出格式
請以以下JSON格式返回知識圖譜：
```json
{
  "relationships": [
    {"source": "<源實體>", "target": "<目標實體>", "relation": "<關係>", "description":"<描述文本>"}
  ]
}
```

每個關係必須包含源實體、目標實體、關係類型和描述文本，其中描述文本應盡可能使用原文的表述。
